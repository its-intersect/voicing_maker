# voicing_maker
Generates 4 part jazz voicings for a given progression

Run voicing_maker.exe

Enter "exit" when asked for progression to leave the program.

Usage:
  Enter chord progression, each chord separated by comma
  Use flat notation only i.e. Gb instead of F#
  Use 'm' for minor, 'M' for major and '7' for dominant chord
  
  Example for ii V I in the key of C would be entered like "Dm,G7,CM"
  

A midi file will be generated after voicings are created
